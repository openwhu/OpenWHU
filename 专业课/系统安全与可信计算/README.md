# 系统安全与可信计算

## 课程介绍

前半学期讲解系统安全的知识，涉及到一些常见的安全模型。

后半学期讲解可信计算知识，涉及到可信基等知识。

### 结课形式

60%平时（实验占大部分）+40%考试（开卷）

## 实验构成

有3次实验，分别是linux的selinux使用与分析（系统安全部分）、TPM2.0使用与分析（可信计算部分）、最后一次实验是大实验，小组作业并安排答辩。

## 考试范围

可能有老师会在考前划考点，建议多打听各个教学班的消息，以及实验做得内容也会考，比如2025-2026最后一题考的是TPM2.0写代码（有20分，实验二里面动手写过）
