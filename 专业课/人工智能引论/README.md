## 2026.1.16更新

人工智能学院成立后，计院就没有人工智能专业了。  
2024级为计院人工智能方向的最后一届学生，24级的课程也是在计院的最后一次开设。  
以下课程介绍基于2024级（25-26学年 第一学期）实际课程内容整理，可能与后续由人工智能学院开设的同名或相近课程存在差异，仅供参考。

> 本人曾修读 2023 级的课程，并在 2024 级课程中担任助教，相关内容结合助教工作经验整理。

### 分数构成

|      项目      | 占比  |
| :------------: | :---: |
| 课堂练习+签到  |  10%  |
|    课后作业    |  20%  |
| 课程实验(3次)  |  20%  |
| 期末考试(开卷) |  50%  |

- 课后作业：以教材配套习题为主，提交后由助教批改。
- 课程实验：根据任课教师要求完成。实验指南与示例代码由助教提供，实验报告亦由助教负责批改。
  - [实验一 实验指南](./实验/25-26/实验一/)
  - [实验三 实验指南](./实验/25-26/实验三/)
- 期末考试：开卷考试，考查重点相对集中，可参考往年试题进行复习。
